# [sample-htmx-spring][repo]

Simple [htmx][htmx] project using spring boot with thymeleaf as backend.

Generated by [spring initializr][initializr].

See [HELP.md][help] for further details.

## Requirements

- maven 3.9
- java 17
- htmx 2.0 (webjar)

## How to build

```bash
./mvnw clean package
```

## How to test


```bash
./mvnw test
```

## How to run

```bash
./mvnw spring-boot:run
```
